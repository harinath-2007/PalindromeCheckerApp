# Palindrome Checker Management System

## 📌 Project Overview

The **Palindrome Checker Management System** is a simple Java-based application that initializes a system for checking whether a given string or number is a palindrome.

This project demonstrates:

* Basic Java program structure
* Use of `main()` method
* Console output
* Application initialization flow

---

## 🚀 Features

* Displays welcome message
* Shows application version
* Confirms system initialization
* Entry point for future palindrome checking functionality

---

## 🛠️ Technologies Used

* Java
* VS Code (Recommended IDE)
* JDK 8 or above

---

## 📂 Project Structure

```
PalindromeCheckerApp/

```

## ▶️ How to Run the Application

### Step 1: Open Project in VS Code

* Open the folder in VS Code.

### Step 2: Compile the Program

Open terminal and run:

```
javac UseCasePalindromeCheckerApp.java
```

### Step 3: Run the Program

```
java UseCasePalindromeCheckerApp
```

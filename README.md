# Palindrome Checker Management System

## 📌 Project Overview

The **Palindrome Checker Management System** is a basic Java application that checks whether a given string is a palindrome.

In this version (UC2), the program:

* Uses a hardcoded string (`"madam"`)
* Reverses the string manually using a loop
* Compares original and reversed strings
* Displays the result in the console

## 🧠 How the Logic Works

1. Store a string (`madam`) in a variable.
2. Reverse the string using a `for` loop.
3. Compare original string with reversed string.
4. If both are equal → It is a palindrome.
5. Otherwise → It is not a palindrome.

## 📘 What is a Palindrome?

A **palindrome** is a word, number, or phrase that reads the same forward and backward.

Examples:

* madam
* level
* 121
* radar
